# Java – Fundamentos e Lógica de Programação

Repositório com mini-projetos desenvolvidos durante meus estudos em Java,
com foco em lógica de programação, estruturas condicionais, entrada de dados
e cálculos matemáticos.

## Conteúdos praticados
- Variáveis e tipos primitivos
- Estruturas condicionais (if/else)
- Entrada de dados com Scanner
- Cálculos matemáticos
- Formatação de saída

## Contexto
Os exercícios foram desenvolvidos como parte da minha formação em
Ciência da Computação e representam minha base em programação antes
do aprofundamento atual em Python.
